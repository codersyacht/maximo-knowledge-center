## Event Monitor

Note:

All queries are executed as db2 commands.

```SQL
db2
```

```CMD
db2 connect to MAXIMO;
```
To execute them in a specific schemma set the following.

```SQL
SET SCHEMA 'MAXIMO';
```

### Enable Data Collection

```SQL
ALTER WORKLOAD SYSDEFAULTUSERWORKLOAD COLLECT ACTIVITY DATA ON ALL MEMBERS WITH DETAILS;
```

Default Values will be N for SYSDEFAULTUSERWORKLOAD and SYSDEFAULTADMWORKLOAD.

Enable data collection for SYSDEFAULTUSERWORKLOAD.

```SQL
ALTER WORKLOAD SYSDEFAULTUSERWORKLOAD COLLECT ACTIVITY DATA ON COORDINATOR WITH DETAILS;
```

Check again if data collection is enabled.

```SQL
SELECT WORKLOADNAME, COLLECTACTDATA FROM SYSCAT.WORKLOADS;
```

### Create Event Monitor

```SQL
CREATE EVENT MONITOR ACTMON FOR ACTIVITIES WRITE TO TABLE;
```

**Check if activity monitoring is created**

```SQL
SELECT TABNAME FROM SYSCAT.TABLES WHERE TABSCHEMA = 'MAXIMO' AND TABNAME LIKE 'ACTIVITY%';
```

### Capture Queries

Enable Activity Monitoring

```SQL
SET EVENT MONITOR ACTMON STATE = 1 ;
```

Run Queries.

Stop Event Monitoring

```SQL
SET EVENT MONITOR ACTMON STATE = 0 ;
```

### Query Results

**General**

```SQL
SELECT COUNT(*) FROM MAXIMO.ACTIVITYSTMT_ACTMON;
```
```SQL
SELECT * FROM MAXIMO.ACTIVITYSTMT_ACTMON ORDER BY STMT_LAST_USE_TIME DESC;
```
```SQL
SELECT ACTIVATE_TIMESTAMP, APPL_ID, CREATOR, STMT_FIRST_USE_TIME, STMT_LAST_USE_TIME, STMT_TEXT 
FROM MAXIMO.ACTIVITYSTMT_ACTMON ORDER BY STMT_LAST_USE_TIME DESC;
```
```SQL
SELECT STMT_TEXT FROM MAXIMO.ACTIVITYSTMT_ACTMON ORDER BY STMT_LAST_USE_TIME DESC;
```

**Performance Related**

```SQL
SELECT count(*) FROM MAXIMO.ACTIVITYMETRICS_ACTMON ;
```
```SQL
SELECT count(*) FROM MAXIMO.ACTIVITYSTMT_ACTMON;
```
```SQL
SELECT count(*) FROM (SELECT S.STMT_TEXT, M.STMT_EXEC_TIME FROM ACTIVITYSTMT_ACTMON S, ACTIVITYMETRICS_ACTMON M 
WHERE S.APPL_ID=M.APPL_ID AND S.UOW_ID=M.UOW_ID AND M.ACTIVITY_ID = S.ACTIVITY_ID ORDER BY  M.STMT_EXEC_TIME DESC);
```
```SQL
SELECT S.STMT_TEXT, M.STMT_EXEC_TIME FROM ACTIVITYSTMT_ACTMON S, ACTIVITYMETRICS_ACTMON M 
WHERE S.APPL_ID=M.APPL_ID AND S.UOW_ID=M.UOW_ID AND M.ACTIVITY_ID = S.ACTIVITY_ID ORDER BY  M.STMT_EXEC_TIME DESC
```

```SQL
DELETE FROM MAXIMO.ACTIVITYVALS_ACTMON;
```
```SQL
DELETE FROM MAXIMO.ACTIVITYMETRICS_ACTMON;
```

### Delete Activity Monitor

Set state to 0 (Disable).

Run the following SQL to list the Event Monitors.

```SQL
SELECT TABNAME FROM SYSCAT.TABLES WHERE TABSCHEMA = 'MAXIMO' AND TABNAME LIKE 'ACTIVITY%';
```

```SQL
SET EVENT MONITOR ACTMONS STATE = 0 ;
```

Delete Activity Monitor.

```SQL
DROP EVENT MONITOR ACTMON;
```
```SQL
DROP TABLE ACTIVITYSTMT_ACTMON;
```
```SQL
DROP TABLE ACTIVITYVALS_ACTMON;
```
```SQL
DROP TABLE ACTIVITYMETRICS_ACTMON;
```
```SQL
DROP TABLE ACTIVITY_ACTMONS;
```

Run the following query to check if activity monitor is deleted.

```SQL
SELECT TABNAME FROM SYSCAT.TABLES WHERE TABSCHEMA = 'MAXIMO' AND TABNAME LIKE 'ACTIVITY%';
```
